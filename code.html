HTML: The backbone of websites


Definition.

Hypertext Markup Language is the programming language used to develop documents that are readable by a web browser

HTML incorporates assisting languages such as Casscading Style Sheets for design and Javascript on functionality

Cool Facts

Head element

Head tags are used to make the defining properties of the website such as the set font, colour, backgroud image and title.

Image element

Image tags are used to make images function and appear on the page. It specifies any properties on the appearance such as its dimensions and borders.

A HREF tags

A HREF attributes are used to locate and display hyperlinks that load websites elsewhere on the Internet.
As it is on the Internet, the specific webpage will need to have its URL clearly referenced for it to redirect the user properly.
